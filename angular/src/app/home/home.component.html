Welcome on the cycling init bot website!

Cycling init bot is a script designed to initialize cycling related items on Wikidata.

This website allows you to access its functionality without having to install anything.

Some news about the tool and info:

The bot has been mainly used for women cycling purposes, the men part may need some improvement.

Lots of actions related to maintenance of Wikidata for cycling are quite repetitive. They can be easily delegated to a script. Here some illustrations.


For Initialization of:
  • National championships
  • Races
  • Team seasons, including national teams
  • Rider
For data importation, this bot used to use UCI as source, but they stopped providing Excel file. We moved to FirstCycling, but they also blocked us, so those functions are not working presently:
  • Race ranking
  • Startlist
  • UCI ranking
  • UCI calendar
Eventually some sorting algorithms are also included:
  • Sorting by date
  • Sorting by name
Your rights will be granted progressively:
  • When first registred and activated, you can ask for bot requests, I will trigger them manually. Thanks for the understanding.
  • When you are granted higher right, the requests will be triggered automatically after sending.

Please note that unfortunately the wikidata API is sometimes slow. It depends on the server load. Some requests can therefore be long or even crash if the maximum lag is reached. I have no influence on this parameter. Requests in the morning tends to be faster.
